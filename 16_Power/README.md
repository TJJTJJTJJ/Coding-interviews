# 数值的整数次方

____

## 题目描述

给定一个 double 类型的浮点数 base 和 int 类型的整数 exponent。求 base的 exponent 次方。

____

## 解法

### v-1.0

乘方用n/2法乘。

exponet可能为负数。

### v-2.0

**指数0不适用于0.**

### v-3.0

**用右移代替了除以2**，

**用位与(& 0x1)代替了判断余数(%)奇偶数**。

边界条件，n=1。
