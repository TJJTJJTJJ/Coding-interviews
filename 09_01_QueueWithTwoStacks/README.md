# 用两个栈实现队列

____

## 题目描述

用两个栈来实现一个队列，完成队列的 Push 和 Pop 操作。 队列中的元素为 int 类型。

____

## 解法

栈和队列互相实现，基本都是两个等效于另一个，这种题没有什么太大难度，只要拿笔实验两三个数字就可以了。

性质：

* push都存在stack1中，
* pop从stack2取，stack2为空时则把stack1的数据全部倒过来。

上述性质同样适用于其他的类似两个等效一个的实验中。

____

## 测试用例

1. 往空的队列里添加、删除元素；
2. 往非空的队列添加、删除元素；
3. 连续删除元素直至队列为空
